# Start the project

#### 1. Clone to repositorie
```git clone https://github.com/zagob/Challenge02-Completed.git```

#### 2. Install dependencies
```yarn``` or ```npm install```

#### 3. Start Server
```yarn server``` or ```npm run server```

#### 4. Start Project
```yarn start``` or ```npm run start```
